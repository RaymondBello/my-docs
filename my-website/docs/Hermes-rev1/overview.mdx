# Introduction
This embedded system is designed to function as the base flight controller responsible for reading sensors and controlling onboard actuators at approx 100Hz. <br/>

## Hardware Specification
 
|  S/N   | Component             | Qty | Description  |
|  --:   | :--                   | :-:  | :--          |
|   1    | [<b>STM32F405RGT6</b>](https://datasheet.lcsc.com/lcsc/1809301224_STMicroelectronics-STM32F405RGT6_C15742.pdf)   |  1  | Main MCU for polling IMU and Radio PPM for control loop |
|   2    | <b>STM32H7xx</b>      |  1  | Secondary MCU handling onboard guidance & navigation algorithms + extras |
